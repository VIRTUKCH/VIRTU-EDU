Start Now를 눌렀을 때
1. 학습 전으로 이동
: 오늘의 학습 계획을 시간과 목표로 작성하게 한다(핵심 개념&질문). 이후 다음을 누르면 학습 중으로 이동한다.
2. 학습 중으로 이동
: 실제로 학습하게 한다. 타이머가 지나고, 본인이 작성했던 목표가 계속 뜬다.
이후 학습 완료를 누르면 학습 후로 이동한다. 타이머가 지나면 시간을 알리는 기능만 한다.
3. 학습 후로 이동
: 본인이 학습 전에 정했던 목표(핵심 개념&질문)를 달성했는지 답변하게 한다.
